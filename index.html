The Couplers

Apa yang dimaksud dengan " The Couplers " ?


Adalah suatu kategori bad smell code yang merujuk pada code yang memiliki ketergantungan yang tinggi antara satu dengan yang lainnya. Ketergantungan yang erat ini dapat membuat code menjadi kaku, sulit untuk dipelihara, dan sulit untuk diubah. Ada 4 macam tipe the couplers, yang akan dijelaskan di website ini.

Feature Envy

Suatu method lebih banyak mengakses data dari objek lain daripada data diri sendiri.

Solusi : memindahkan method ke class yang datanya digunakan.

Inappropriate Intimacy

Satu class menggunakan internal fields dan methods dari class lain.

Solusi : mengubah method pada class tersebut.

Message Chain

Terjadi series of calls yang seperti $a -> b() -> c() -> d().

Solusi : membuat sebuah method untuk mengurangi message chain.

Middle Man

Sebuah method yang tugasnya hanya memanggil method lain (entah dalam class sendiri atau milik class lain).

Solusi : Menghapus method tersebut dan user langsung memanggil method yang diinginkan.
